## The CAVEGui Project


Link: [http://www.evl.uic.edu/cavern/cavegui](http://www.evl.uic.edu/cavern/cavegui)


![image](/images/cavegui_edit.gif){:style="max-width: 100%"}

CAVEGui: is a graphical user interface tool that provides interaction with a CAVE application, sensor rotation and translation along with controller navigation and buttons, from a desktop environment using a standard mouse.
