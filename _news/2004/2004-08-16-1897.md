## Electro


Link: [http://www.evl.uic.edu/rlk/electro](http://www.evl.uic.edu/rlk/electro)

Driving demo displayed on Varrier Autostereoscopic Display

![image](https://www.evl.uic.edu/output/originals/driving-varrier-thumb.jpg-srcw.jpg){:style="max-width: 100%"}
Credit: R. Kooima, EVL

Electro is an application development environment designed for use on both cluster-driven tiled displays and desktop systems.<br><br>

Electro is based on the MPI process model and is bound to the Lua programming language.<br><br>

With support for 3D graphics, 2D graphics, audio, and input handling, Electro provides an easy-to-use scripting system for interactive applications spanning multiple processors and displays.<br><br>

Electro supports Linux, Windows 2K / XP, and Mac OS.
