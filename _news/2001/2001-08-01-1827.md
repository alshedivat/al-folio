## ImmersaView


Link: [http://www.evl.uic.edu/cavern/agave/immersaview](http://www.evl.uic.edu/cavern/agave/immersaview)


![image](/images/immersaview_logo05.jpg-srcw.jpg){:style="max-width: 100%"}
Credit: CAVERN Group / EVL

Immersaview is an Open Inventor / VRML model viewer for the GeoWall / AGAVE. The data can be a single Inventor model or a flipbook of models that you can animate or single-step through.<br><br>

This software uses <a href='http://www.coin3d.org'>Coin3D</a> and <a href='http://www.opengl.org'>GLUT</a>, used to compile ImmersaView from scratch.<br><br>

ImmersaView is currently supported in the <a href='http://linux/downloadLinux.html'>Linux</a>, <a href='http://win/downloadWin.html'>Windows</a> and <a href='http://win/downloadMac.html'>Macintosh</a> environments.
