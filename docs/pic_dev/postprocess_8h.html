Gathers a PETSc vector or a DMSwarm field, prepares VTKMetaData, and writes to a VTK file.

This function dynamically handles both Eulerian (.vts) and Lagrangian (.vtp) data:

  • If the field exists in DMSwarm, it extracts it into a PETSc Vec.
  • If already a Vec, it directly processes it.
  • For .vtp files, it also gathers particle positions (DMSwarmPICField_coor).
  • For .vts, only the field is gathered (structured grid).
Parameters
[in]userPointer to user context (contains swarm and Eulerian fields).
[in]fieldNameName of the field (e.g., "Ucat" for vts, "Velocity" for vtp).
[in]coorfieldNameName of the coordinate field used (e.g., "positions" or "pos_phy".
[in]timeIndexTimestep index for filename (e.g., "Ucat_00010.vts").
[in]outExtFile extension ("vts" for structured, "vtp" for particles).
[in]prefixFile prefix(directory to store file in).
[in]commMPI communicator (usually PETSC_COMM_WORLD).
Returns
PetscErrorCode 0 on success, or an error code on failure.

Writes particle (Lagrangian) data fields to separate .vtp files.
