371/**
372 * @brief Parses the boundary conditions file to configure the type, handler, and
373 * any associated parameters for all 6 global faces of the domain.
374 *
375 * This function performs the following steps:
376 * 1. On MPI rank 0, it reads the specified configuration file line-by-line.
377 * 2. It parses each line for `<Face> <Type> <Handler> [param=value]...` format.
378 * 3. It validates the parsed strings and stores the configuration, including a
379 * linked list of parameters, in a temporary array.
380 * 4. It then serializes this configuration and broadcasts it to all other MPI ranks.
381 * 5. All ranks (including rank 0) then deserialize the broadcasted data to populate
382 * their local `user->boundary_faces` array identically.
383 * 6. It also sets legacy fields in UserCtx for compatibility with other modules.
384 *
385 * @param[in,out] user The main UserCtx struct where the final configuration
386 * for all ranks will be stored.
387 * @param[in] bcs_input_filename The path to the boundary conditions configuration file.
388 * @return PetscErrorCode 0 on success, error code on failure.
389 */
390PetscErrorCode ParseAllBoundaryConditions(UserCtx *user, const char *bcs_input_filename);
